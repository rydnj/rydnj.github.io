# ryandenny.dev

My personal portfolio site. Built with [Astro](https://astro.build/) and [Tailwind CSS v4](https://tailwindcss.com/), based on the fork from [DevPortfolio](https://github.com/RyanFitzgerald/devportfolio) template done by Ryan Fitzgerald.

## Live

[https://rydnj.github.io](https://rydnj.github.io)

## Local Development

```bash
git clone https://github.com/rydnj/rydnj.github.io.git
cd rydnj.github.io
npm install
npm run dev
```

## Built With

- [Astro](https://astro.build/) — Static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) — Utility-first CSS
- [Tabler Icons](https://tabler.io/icons) — Icons
- TypeScript — Configuration
